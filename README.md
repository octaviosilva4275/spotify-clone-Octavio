# Spotify Imersão - Clone

Este é um projeto de um clone do **Spotify**, criado durante a imersão em Front-End. O objetivo é replicar o layout e algumas funcionalidades básicas do Spotify, como a navegação entre playlists, a busca de músicas e artistas, e a visualização de conteúdo como playlists e cards.

## Estrutura do Projeto

O projeto é composto pelas seguintes seções:

- **Sidebar**: A barra lateral contém links para a página inicial, a busca, a biblioteca pessoal e a opção de criar uma playlist.
- **Header**: O cabeçalho possui opções para navegação, como botões de navegação lateral (esquerda e direita), campo de busca e botões de login/inscrição.
- **Main**: Exibe as playlists e o conteúdo principal, como imagens de capas de álbuns e músicas, com a capacidade de navegar entre diferentes categorias de playlists.
- **Footer**: Oferece informações sobre o plano Premium, com uma opção para inscrever-se gratuitamente.

## Funcionalidades

- **Exibição de Playlists**: O site exibe diversas playlists e categorias, como "Boas festas", "Lançamentos", "MPB", "Funk", "Indie", entre outras.
- **Busca de Conteúdo**: Um campo de busca permite ao usuário procurar por músicas, artistas e playlists.
- **Criação de Playlist**: O site oferece a possibilidade de criar a primeira playlist do usuário.
- **Navegação**: O usuário pode navegar entre as playlists utilizando botões de setas para a direita e esquerda.
- **Modo Responsivo**: O layout foi projetado para ser responsivo, adaptando-se a diferentes tamanhos de tela, com uso de CSS media queries.

## Tecnologias Utilizadas

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização, incluindo o uso de reset e variáveis CSS para facilitar a personalização.
- **FontAwesome**: Ícones para a navegação e outros elementos interativos.
- **JavaScript/jQuery**: Interatividade com a página, como a navegação entre playlists.

## Como Usar

1. Clone o repositório ou faça o download dos arquivos.
2. Abra o arquivo `index.html` em um navegador para visualizar o site.
3. Personalize as imagens e as playlists conforme necessário.
